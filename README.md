# ChangeRosetta
use python script or CLI to change Rosetta2 with architecture(x86_64 or arm64)

# usage
* use python 
  1. go to the python file folder 
  2. run `config_rosetta.py x86_64` to change the checkbox value to check
  ### note: default architecture value is `x86_64`, you can use `config_rosetta.py arm64` to uncheck this value, the default app is `Xcode`
  
* use CLI
  1. go to the `ChangeRosetta` file folder 
  2. run this cmd in Terminal `ChangeRosetta <app_path> <x86_64 or arm64>`
